## 生成hexo
  &emsp; &emsp; 需要先安裝node.js和git
<!-- more -->
### 安裝(install) Hexo
```js 
npm install -g hexo-cli
``` 

### 興建 Hexo
```js
hexo init <folder> //<folder>要新增的名稱
cd <folder>
npm install
```
